# Plotzy — ملف السياق الكامل للمشروع

## وصف المشروع
Plotzy منصة كاملة لكتابة ونشر الكتب. تتيح للكتّاب كتابة الفصول، تصميم الأغلفة، نشر الكتب وبيعها، مع دعم الذكاء الاصطناعي.

---

## Stack التقني

| الطبقة | التقنية |
|--------|---------|
| Frontend | React 18 + Vite + TypeScript + TailwindCSS |
| Backend | Express 5 + tsx (port 5000) |
| Database | PostgreSQL + Drizzle ORM |
| Auth | Passport.js (local + Google OAuth + Apple) |
| Payments | Stripe |
| AI | OpenAI API (gpt-4o-mini) |
| Monorepo | pnpm workspace |

## قواعد حرجة — اقرأها قبل أي تعديل

### 1. storage.ts — CRLF line endings
الملف يستخدم `\r\n` — دائماً استخدم `node -e` bash scripts للتعديل عليه.

### 2. Auth middleware
لا توجد دالة `requireAuth` — الحماية inline:
```typescript
if (!req.isAuthenticated()) return res.status(401).json({ message: "Unauthorized" });
```

### 3. Gutenberg API
`gutendex.com` محجوب في Replit. البروكسي يستخدم:
`https://www.gutenberg.org/ebooks/search.opds/` مع XML regex parsing.

### 4. Routing Architecture
Browser → Replit proxy (port 80) → api-server (port 8080) → Express (port 5000)

### 5. صفحات بدون Layout (مقصود — لا تغيّر)
`gutenberg-reader.tsx` / `chapter-editor.tsx` / `read-book.tsx` / `cover-designer.tsx`

## تفضيلات التصميم — مهم جداً لا تخالفها

- **لا** ألوان violet أو rainbow
- **لا** إحصاءات وهمية
- **لا** italic في نص الكتاب
- **لا** em-dashes في نص الكتاب
- **لا** horizontal rules زخرفية
- **لا** spine line في منتصف الكتاب
- الأسلوب: Apple aesthetic، أسود/داكن، monochromatic
- الخط: SF Pro Display

## آخر التعديلات المنجزة في هذه الجلسة

### 1. i18n — ترجمة كاملة للـ nav
- أُضيفت مفاتيح الترجمة في `i18n.ts` لـ 14 لغة:
  `navGuide, navMarketplace, navCommunity, navPublicDomain, navPro, navSupport, signIn, changeDisplayName, signOut, noNameSet`
- `layout.tsx` يستدعي `t()` لكل عناصر الـ nav والـ dropdown
- تم إضافة RTL `dir` attribute على الـ root wrapper

### 2. Book Spread — الصفحة الرئيسية (`home.tsx`)
قيم التصميم الحالية:
```javascript
PAGE_BG   = "#F2ECD8"
PAGE_TEXT = "#1a1209"
BOOK_FONT = "'Merriweather', 'Lora', Georgia, serif"
```
- Running header: small-caps، حجم 0.62rem، opacity 0.45
- Chapter label: italic، حجم 0.78rem، color `#6b5a3e`
- Chapter heading: bold، حجم 1.32rem، color `#1a1209`
- Body text: حجم 0.82rem، line-height 1.78، justified
- Drop cap على أول حرف في الصفحة اليسرى
- Page numbers: `— 5 —` style، حجم 0.7rem
- Padding: 56px جانبين، 42px أسفل، 28px أعلى
- `marginTop: 18` على container النص (تحت العنوان)
- **لا** em-dashes في النص — تم استبدالها بفواصل ونقاتين

### 3. Container Scroll Animation (`container-scroll-animation.tsx`)
الأبعاد الحالية:
```javascript
// الحاوية الخارجية
h-[52rem] md:h-[68rem]  padding: p-2 md:p-4

// الحاوية الداخلية
py-2 md:py-10

// بطاقة الكتاب
h-[36rem] md:h-[46rem]

// الأنيميشن
rotate    = useTransform(scrollYProgress, [0, 1], [20, 0])
scale     = useTransform(scrollYProgress, [0, 1], [1.05, 1])
translate = useTransform(scrollYProgress, [0, 1], [0, -100])
```

### 4. Feature Phrases Section (`home.tsx`)
```javascript
marginTop: "-60px"  // تم تعديله من -140px لتجنب التداخل مع الكتاب
```
