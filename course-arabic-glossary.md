# Arabic Translation Glossary — Plotzy Writing Course

**Status:** LOCKED 2026-05-07. Five judgment calls reviewed by Faris; all locked terms below match his decisions. Use these throughout Module 1 onward.

This glossary fixes the Arabic translation of every recurring writing-craft term across the 27 lessons + 70 quiz questions. Lock these once and use them everywhere — readers shouldn't see a term as `البطل` in lesson 1 and `الشخصية الرئيسية` in lesson 5.

**Voice register:** Modern Standard Arabic (فصحى معاصرة), educational tone, accessible to a non-specialist literary audience. Avoid archaic terms; avoid colloquial. Match the English voice (direct, occasionally poetic, never academic-stiff).

**Punctuation:** Use Arabic punctuation marks (، ؛ ؟) consistently with formal Arabic prose. **No em-dashes (—)** — same 4-pattern guide as the English em-dash batch (parenthetical → commas, emphasis → colon, break → period, range → "إلى").

## Section 1 — Core narrative concepts

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| story | قصة | Universal. Not رواية (which is "novel"). |
| plot | الحبكة | Standard literary-critical term. |
| premise | المُنطَلق | Lit. "the launching point." Locked. |
| theme | الموضوع المحوري | Lit. "the central topic." Locked. |
| genre | النوع الأدبي | Standard. |
| conflict | الصراع | Universal. |
| tension | التوتر | Standard. |
| pacing | الإيقاع | Standard. The same word as "rhythm" — context disambiguates. |
| stakes | الرهانات | Lit. "the wagers." Captures "what's at stake" naturally. |

## Section 2 — Character terms

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| protagonist | البطل | Default. Where gender matters: البطلة. |
| antagonist | الخصم | Alternative: الشخصية المعارضة. الخصم is shorter and more idiomatic. |
| supporting character / cast | الشخصيات المساندة | Lit. "supporting." The English "cast" is best dropped; Arabic doesn't carry the theatrical metaphor naturally. |
| character arc | قوس الشخصية | Direct calque, widely understood in Arabic crit. |
| character wound | جرح الشخصية | Truby/Mamet term. Arabic كلمة جرح captures both physical and psychological. |
| want | الرغبة الظاهرة | Lit. "the overt desire." The simple الرغبة is too generic — pairing with الحاجة الجوهرية below makes the contrast land. |
| need | الحاجة الجوهرية | Lit. "the essential need." Forms the canonical want/need pair. |
| backstory | الخلفية الشخصية | Lit. "the personal background." Avoid التاريخ which sounds historical. |
| voice | الصوت السردي | Lit. "the narrative voice." Distinguishes from physical voice. |

## Section 3 — Structure terms

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| three-act structure | البنية ثلاثية الفصول | Standard descriptive translation. |
| hero's journey | رحلة البطل | Standard. Specifically Campbell's monomyth. |
| beat sheet | جدول المشاهد المحورية | Lit. "key-scenes table." Faris prefers جدول over ورقة because beat sheets are table-like grids, not loose sheets. Locked. |
| inciting incident | الحدث المُحرّض | Standard. الحدث المُفجِّر is also used. |
| midpoint reversal | الانعطافة الوسطى | Lit. "the middle turn." |
| climax | الذروة | Universal. |
| resolution | الخاتمة | Standard. الحل is also used; خاتمة is more literary. |
| rising action | التصاعد الدرامي | Lit. "dramatic ascent." |
| falling action | الانحدار / النزول | Both work. |

## Section 4 — Craft moves

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| show, don't tell | أَظهِر ولا تَحكِ | Direct calque. Standard in modern Arabic creative-writing crit. Locked. |
| point of view (POV) | وجهة النظر السردية | Standard. |
| first person | ضمير المتكلم | Grammatically standard. |
| third person | ضمير الغائب | Grammatically standard. |
| foreshadowing | التمهيد | Lit. "preparing the ground." الإيحاء المُسبَق is more literal but clunkier. |
| subtext | ما تحت السطح | Lit. "what's beneath the surface." Reads naturally; avoids the academic المضمون الخفي. |
| dialogue tag | علامة الحوار | Direct, clear. |
| sensory writing | الكتابة الحسّية | Lit. "sense-based writing." Standard. |
| world-building | بناء العالم | Direct calque, widely accepted. |
| setting | المكان والزمان | Lit. "place and time." Default rendering. Locked. **Context note:** in world-building lessons specifically, expand to **المكان والزمان والأجواء** when atmosphere is part of the discussion. Do NOT substitute البيئة السردية — too academic for the course voice. |

## Section 5 — Process terms

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| first draft | المسودة الأولى | Standard. |
| revision | المراجعة | Distinguished from التحرير (editing) below. |
| editing | التحرير | Standard. |
| writer's block | جُمود الكاتب | Lit. "writer's frozenness." Alternative: انسداد الكتابة. جمود الكاتب is the more idiomatic choice in Arabic crit. |
| outline | المخطط | Standard. |
| beta reader | القارئ التجريبي | Lit. "the experimental reader." There is no Arabic literary tradition for this concept; descriptive rendering wins. |

## Section 6 — Publishing terms

| English | Proposed Arabic | Notes / rationale |
|---|---|---|
| self-publishing | النشر الذاتي | Standard. |
| traditional publishing | النشر التقليدي | Standard. |
| literary agent | الوكيل الأدبي | Standard. |
| query letter | رسالة عرض المخطوطة | Lit. "manuscript-pitch letter." Arabic publishing doesn't have a tight equivalent for "query"; descriptive is clearest. |
| cover | الغلاف | Standard. |
| blurb | الملخص الترويجي | Lit. "promotional summary." There's no Arabic word for "blurb"; this is the standard rendering. |
| audience | القرّاء | Lit. "the readers." الجمهور is broader (audience-as-public); for writing-craft, القرّاء is more precise. |
| platform (author platform) | الحضور الإلكتروني | Lit. "online presence." The English "platform" in publishing-business sense is hard in Arabic; this captures the meaning. |

## Translation conventions (apply throughout)

1. **Numbers:** Use Western numerals (0–9) consistent with the English source. Arabic-Indic numerals (٠–٩) are correct in formal Arabic but less universal across the Arabic web; Western numerals match the platform's existing UI.
2. **Person and tense:** Match the English source's choice of person (the lessons mostly use second-person "you," which renders naturally as أنت / `tu`-form imperatives).
3. **English literary references** (Macbeth, Ahab, *War and Peace*, etc.): **transliterate, do NOT substitute** with Arabic equivalents. Per Faris's Phase A decision.
4. **Latin punctuation in markdown structure** (`#`, `##`, `*`, `**`, `>`, `[link](url)`): preserve verbatim. Only the prose translates.
5. **Code blocks and exercises** (e.g., "Exercise 1.1: …"): translate the prose; preserve the numbering.
