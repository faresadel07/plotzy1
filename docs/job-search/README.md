# Job Search Kit — Faris Hamdan

Everything you need to land a remote full-stack engineering job in

## The files

1. [`01_COVER_LETTERS.md`](./01_COVER_LETTERS.md) — three cover letter
   variants (AI, SaaS startup, established company) plus a follow-up

   companies and 13 job boards sorted by realistic chance of a reply,

4. [`04_DAILY_PLAYBOOK.md`](./04_DAILY_PLAYBOOK.md) — the exact

   apply in 90 seconds, how to prep for the first call, realistic
